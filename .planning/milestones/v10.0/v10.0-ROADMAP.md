## Visão Geral

Separar o `frontend/` monolítico (Expo + react-native-web) em dois projetos independentes otimizados para suas plataformas:

- **`mobile-frontend/`** — React Native + Expo SDK 51 (iOS + Android)
- **`web-frontend/`** — Next.js 15 App Router (SEO-first, SSR/SSG)
